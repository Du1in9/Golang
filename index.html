一、Go 教程

1.1 Go 概述

1.1.1 Go 语言特色

  • 简洁、快速、安全

  • 并行、有趣、开源

  • 内存管理、数组安全、编译迅速

1.1.2 Go 语言用途

Go 语言被设计成一门应用于搭载 Web 服务器，存储集群或类似用途的巨型中央服务器的系统编程语言。

对于高性能分布式系统领域而言，Go 语言无疑比大多数其它语言有着更高的开发效率。

它提供了海量并行的支持，这对于游戏服务端的开发而言是再好不过了。

1.1.3 第一个 Go 程序

接下来我们来编写第一个 Go 程序 hello.go（Go 语言源文件的扩展是 .go），代码如下：

​x
package main


import "fmt"


func main() {
    fmt.Println("Hello, World!")
}

Go 语言的基础组成有以下几个部分：包声明、引入包、函数、变量、语句 & 表达式、注释。

  • 要执行 Go 语言代码可以使用 go run 命令：go run hello.go

  • 还可以使用 go build 命令来生成二进制文件：go build hello.go

1.2 Go 基础语法

Go 程序可以由多个标记组成，可以是关键字，标识符，常量，字符串，符号。如以下 GO 语句由 6 个标记组成：

6 个标记是：fmt 、. 、Println 、( 、 "Hello, World!" 、)。

如果你打算将多个语句写在同一行，它们则必须使用分号人为区分，但在实际开发中我们并不鼓励这种做法。以下为两个语句：

2.1 Go 数据类型

Go语言的数据类型可以分为几个主要类别：基础数据类型、复合数据类型、指针、接口和错误类型。

2.1.1 基础数据类型

  1. 整型 (int, int8, int16, int32, int64, uint, uint8, uint16, uint32, uint64, uintptr)

  2. 浮点型 (float32, float64)

  3. 复数型 (complex64, complex128)

  4. 布尔型 (bool)

  5. 字符串型 (string)

2.1.2 复合数据类型

  1. 数组 ([ ]T)：固定长度的序列。

    xxxxxxxxxx
    var arr [5]int = [5]int{1, 2, 3, 4, 5}

  2. 切片 ([]T)：动态数组，可以增长和缩小。

    xxxxxxxxxx
    var slice []int = []int{1, 2, 3, 4, 5}

  3. 映射 (map[K]V)：键值对集合，键必须是可比较的类型。

    xxxxxxxxxx
    var m map[string]int = map[string]int{"age": 25, "score": 90}

  4. 结构体 (struct)：由一系列字段组成，字段可以是任意类型。

    xxxxxxxxxx
    type Person struct {
        Name string
        Age  int
    }
    var p Person = Person{Name: "John", Age: 30}

  5. 通道 (chan T)：用于在不同的 goroutine 之间进行通信。

    xxxxxxxxxx
    ch := make(chan int)
    go func() {
        ch <- 1                 // 发送数据
    }()
    v := <-ch                   // 接收数据

2.1.3 指针

  • 指针类型表示内存地址。

xxxxxxxxxx
var i int = 10
pi := &i                        // pi是指向i的指针
fmt.Println(*pi)                // 输出10

2.1.4 接口

  • 接口是定义了一组方法的类型。

xxxxxxxxxx
type Reader interface {

2.1.5 错误类型

  • error是内置接口，用于表示错误。

xxxxxxxxxx
func Divide(a, b int) (int, error) {
    if b == 0 {
        return 0, errors.New("divide by zero")
    }

2.2 Go 变量

在 Go 语言中，变量是用来存储数据值的容器。Go 是静态类型语言，这意味着每个变量都有明确的类型，且必须在使用前声明。

2.2.1 声明变量

使用var关键字声明变量，语法如下：

xxxxxxxxxx
var age int = 30

2.2.2 短变量声明

Go语言还提供了一种快捷声明变量的方式，称为短变量声明。它使用:=操作符，编译器会根据右侧的值自动推断变量的类型。

xxxxxxxxxx
name := "Jack"

2.2.3 变量作用域

  • 局部变量：在函数内部声明的变量，只能在该函数内部访问。

  • 全局变量：在函数外部声明的变量，可以被程序中的任何函数访问。

xxxxxxxxxx
var globalVar int = 10                  // 全局变量


func main() {
    localVar := 5                       // 局部变量
}

2.2.4 变量初始化

如果声明时没有指定初始值，不同类型的变量会有不同的零值：

整型：0，浮点型：0.0，布尔型：false，字符串：""，指针：nil，接口：nil。

2.2.5 常量

常量是不可变的变量，使用const关键字声明。

xxxxxxxxxx
const MaxInt int = 1 << 63 - 1

2.2.6 类型转换

可以使用类型转换将一个变量的值转换为另一种类型。

xxxxxxxxxx
var floatVar float64 = 3.14
var intVar int = int(floatVar)          // 类型转换

2.2.7 多重赋值

Go 语言支持多重赋值，可以一次性给多个变量赋值。

xxxxxxxxxx
a, b := 1, 2
fmt.Println(a, b)                       // 输出：1 2


a, b = b, a
fmt.Println(a, b)                       // 输出：2 1

2.2.8 匿名变量

在某些情况下，你可能需要忽略某些值，可以使用下划线_作为匿名变量。

xxxxxxxxxx
_, err := someFunction()
if err != nil {
    fmt.Println("Error occurred")
}

2.2.9 变量可见性规则

  • 如果变量名以大写字母开头，则在包外可见（导出的）。

  • 如果变量名以小写字母开头，则只在包内可见（未导出的）。

xxxxxxxxxx
var PublicVar int = 10                  // 导出的变量


func main() {
    privateVar := 5                     // 未导出的变量
    // fmt.Println(PublicVar)           // 可以访问
    // fmt.Println(privateVar)          // 编译错误
}

2.3 Go 常量

常量（Constants）在 Go 语言中是固定不变的值，它们在编译时就已经确定。

2.3.1 常量特点

  1. 类型安全：常量是有类型的。

  2. 编译时常数：常量的值在编译时就已经确定。

  3. 不可更改：一旦常量被声明，其值就不能被修改。

  4. 可以是字符、字符串、布尔值或数字：常量可以是任何基本数据类型的值。

2.3.2 声明常量：

  1. 声明整数常量：

xxxxxxxxxx
const MaxInt int = 100          // 显式指定类型
const MinInt = -100             // 类型由编译器推断

  1. 声明浮点数常量：

xxxxxxxxxx
const Pi float64 = 3.14159

  1. 声明布尔常量：

xxxxxxxxxx
const IsDebugging bool = true

  1. 声明字符常量：

xxxxxxxxxx
const UnicodeChar rune = 'A'

  1. 声明字符串常量：

xxxxxxxxxx
const Greeting string = "Hello, World!"

2.3.3 常量组

Go 语言还支持将常量组织成一组，称为常量组。这可以通过省略const关键字并用大括号{}包围来实现。

xxxxxxxxxx
const (
    Byte   = 1
    Kilobyte = Byte * 1024
    Megabyte = Kilobyte * 1024
    Gigabyte = Megabyte * 1024
)

在这个例子中，Byte、Kilobyte、Megabyte和Gigabyte都是常量，并且它们之间的关系是乘法关系。

2.3.4 使用场景

变量可以在程序运行时改变其值。常量的值在编译时确定，并且在程序的整个生命周期内不能改变。

常量通常用于定义程序中不会改变的值，比如：

  • 配置参数，如端口号、超时时间。

  • 数学常数，如圆周率 π。

  • 错误代码，如 HTTP 状态码。

  • 枚举值，如星期几或月份。

使用常量的好处是可以提高代码的可读性和可维护性。如果需要修改一个值，你只需要在一个地方修改，而不是在代码的多个地方。

Go语言提供了丰富的运算符，用于执行算术、比较、逻辑、位运算等操作。

  2. 逻辑与 (&&) 示例: var isBothTrue bool = true && true

  4. 按位取反 (^) 示例: var result int = ^12 // -1 ^ 1100 = 0011

  5. 左移 (<<) 示例: var result int = 2 << 2 // 10

2.4.5 赋值运算符

  1. 等于 (=) 示例: a = 10

  2. 加等于 (+=) 示例: a += 5

  3. 减等于 (-=) 示例: a -= 5

  4. 乘等于 (*=) 示例: a *= 5

  5. 除等于 (/=) 示例: a /= 4

  6. 取模等于 (%=) 示例: a %= 4

  7. 按位与等于 (&=) 示例: a &= 5

  9. 按位异或等于 (^=) 示例: a ^= 5

  10. 左移等于 (<<=) 示例: a <<= 2

三、Go 控制结构

3.1 Go 条件语句

Go 语言的条件语句用于基于特定条件执行不同的代码块。Go 语言中主要有两种条件语句：if 语句和 switch 语句。

3.1.1 if 语句

  • if 语句用于基于一个布尔表达式的值来执行代码。如果表达式的结果为 true，则执行 if 块中的代码。

xxxxxxxxxx
if score > 70 {
    fmt.Println("Pass")
} else {
    fmt.Println("Fail")
}

  • Go语言还允许 if 语句执行初始化语句，这通常用于声明局部变量。

xxxxxxxxxx
if v := 10; v > 5 {
    fmt.Println("v is greater than 5")
} else {
    fmt.Println("v is less than or equal to 5")
}

3.1.2 switch 语句

  • switch 语句用于基于不同的情况执行不同的代码块。它比多个 if-else 语句更清晰。

xxxxxxxxxx
switch grade {
    case "A":
        fmt.Println("Excellent")
    case "B":
        fmt.Println("Good")
    case "C":
        fmt.Println("Average")
    case "D":
        fmt.Println("Below Average")
    default:
        fmt.Println("Fail")
    }

  • Go语言的 switch 语句可以不使用表达式，而是直接对一个条件进行多个测试。

xxxxxxxxxx
switch {
    case a > b:
        fmt.Println("a is greater than b")
    case a == b:
        fmt.Println("a is equal to b")
    default:
        fmt.Println("a is less than b")
    }

3.2 Go 循环语句

Go语言提供了几种循环控制语句，允许代码重复执行直到满足特定的条件。这些循环控制语句包括 for 循环和 range 循环。

3.2.1 for 循环

  • for 循环是Go语言中最基本的循环结构，可以执行一个或多个初始化语句，然后检查一个条件，最后执行一个或多个迭代语句。

xxxxxxxxxx
for i := 0; i < 10; i++ {
    fmt.Println(i)
}

  • 如果省略了初始化、条件和迭代语句，for 循环就会变成一个无限循环。

xxxxxxxxxx
for {
    fmt.Println("Hello, world!")
    time.Sleep(1 * time.Second)         // 暂停1秒
}

  • Go 语言允许在循环前添加标签，并通过 break 语句退出带有标签的循环。

xxxxxxxxxx
Loop1:
for i := 0; i < 3; i++ {
    for j := 0; j < 3; j++ {
        if i*j == 0 continue Loop1
        fmt.Println("i =", i, "j =", j)
    }
}

3.2.2 range 循环

  • range 循环用于遍历数组、切片、映射（map）和字符串。

xxxxxxxxxx
numbers := []int{1, 2, 3, 4, 5}


for index, value := range numbers {
    fmt.Printf("Index: %d, Value: %d\n", index, value)
}
xxxxxxxxxx
str := "Hello, World!"


for index, value := range str {
    fmt.Printf("Index: %d, Value: %c\n", index, value)
}
xxxxxxxxxx
dictionary := map[string]int{
    "apple":  2,
    "banana": 5,
    "cherry": 8,
}


for key, value := range dictionary {
    fmt.Printf("Key: %s, Value: %d\n", key, value)
}

 

 

四、Go 函数与作用域

4.1 Go 函数

Go 语言中的函数是一段可以被多次调用的代码块，用于执行特定的任务。

4.1.1 函数定义

  • Go 语言的函数定义非常严格，要求函数的参数和返回值都必须明确指定类型。

xxxxxxxxxx
func add(x int, y int) int {
    return x + y
}

4.1.2 多返回值

  • Go 语言的函数可以返回多个值。

xxxxxxxxxx
func calculate(x, y int) (int, int) {
    sum := x + y
    difference := x - y
    return sum, difference
}

4.1.3 变长参数

  • 函数可以接受任意数量的参数。

xxxxxxxxxx
func sum(nums ...int) int {
    total := 0
    for _, num := range nums {
        total += num
    }
    return total
}

4.1.4 匿名函数

  • Go 语言支持匿名函数，也称为 Lambda 表达式。

xxxxxxxxxx
func() {
    fmt.Println("Hello, anonymous function!")       // 1. Hello, anonymous function!
}()


greet := func(name string) {
    fmt.Println("Hello,", name)
}
greet("World")                                      // 2. Hello, World

4.1.5 递归函数

  • Go 语言的函数可以调用自身，实现递归。

xxxxxxxxxx
func factorial(n int) int {
    if n == 0 return 1
    return n * factorial(n-1)
}

4.1.6 函数作为参数

  • Go 语言允许将函数作为参数传递给其他函数。

xxxxxxxxxx
func applyFunction(x int, f func(int) int) int {
    return f(x)
}


func double(x int) int {
    return x * 2
}


func main() {
    result := applyFunction(5, double)
    fmt.Println("Result:", result)
}

4.1.7 函数作为返回值

  • Go 语言允许函数返回另一个函数。

xxxxxxxxxx
func greaterThan(n int) func(int) bool {
    return func(x int) bool {
        return x > n
    }
}


func main() {
    // 创建一个函数，该函数返回一个大于5的布尔值
    greaterThan5 := greaterThan(5)
    fmt.Println(greaterThan5(3))        // 输出 false
    fmt.Println(greaterThan5(6))        // 输出 true
}

4.2 Go 变量作用域

变量的作用域（Scope）定义了变量可以被访问的代码区域。理解变量的作用域对于编写清晰和可维护的代码非常重要。

Go 语言中的变量作用域主要包括全局作用域、局部作用域、代码块作用域和函数作用域。

4.2.1 全局作用域

  • 全局变量在整个程序中都可以访问。

xxxxxxxxxx

func someFunction() {
    fmt.Println(globalVar)              // 可以访问全局变量
}

4.2.2 局部作用域

  • 局部变量仅在声明它们的函数或代码块内部可见。

xxxxxxxxxx

}

    // fmt.Println(localVar1)           // 报错
    var localVar2 int = 40

}

4.2.3 代码块作用域

  • 在 Go 语言中，代码块（例如 if、for、switch 等）内部声明的变量仅在该代码块内部可见。

xxxxxxxxxx

        var blockVar int = 50
        fmt.Println(blockVar)           // 输出 50
    }
    // blockVar 在这里是不可见的
}

4.2.4 函数作用域

  • 函数参数和在函数内部声明的变量具有函数作用域，它们仅在函数内部可见。

xxxxxxxxxx
func main() {
    var funcVar int = 60
    fmt.Println(funcVar)                // 输出 60


    funcVar = someFunction(funcVar)
    fmt.Println(funcVar)                // 输出 120
}


func someFunction(x int) int {
    return x * 2
}

4.2.5 匿名函数中的变量作用域

  • 匿名函数可以访问其外部作用域的变量。

xxxxxxxxxx
func main() {
    x := 10
    increment := func() {x = x + 1}
    increment()
    fmt.Println(x)                      // 输出 11
}

4.2.6 闭包中的变量作用域

  • 闭包是指能够读取或修改另一个函数内部变量的函数。

xxxxxxxxxx
func main() {
    x := 10
    increment := func() (func() int, int) {
        return func() int {return x + 1}, x
    }


    nextValue, currentValue := increment()
    fmt.Println(nextValue())            // 输出 11
    fmt.Println(currentValue)           // 输出 10
}

4.2.7 命名冲突

  • 在 Go 语言中，如果局部变量和全局变量同名，局部变量会遮蔽全局变量。

xxxxxxxxxx
var globalVar int = 5


func main() {
    var globalVar int = 10
    fmt.Println(globalVar)              // 输出 10
}

5.1 Go 数组

Go 语言中的数组是一种基础数据结构，用于存储一组具有相同类型的元素。

1. 数组的定义

  • 在 Go 语言中，数组是一种固定长度的数据结构，声明时需要指定数组的长度和元素类型。

xxxxxxxxxx

  • 可以直接通过索引来修改数组中的元素。

  • 数组的长度是固定的，可以在编译时确定。

xxxxxxxxxx
fmt.Println(len(numbers))                   // 输出数组的长度

6. 数组的遍历

  • 可以使用 for 循环或 for range 循环来遍历数组。

7. 多维数组

8. 数组是值类型

5.2 Go 指针

Go 语言中的指针是指向变量内存地址的变量，它们用于直接操作内存地址，从而实现高效的数据共享、修改和传递。

5.2.1 指针的声明和初始化

  • 在 Go 中，声明指针使用星号（*）标识，初始化指针时，为其赋予一个变量的内存地址。

xxxxxxxxxx

p = &i                                      // 将变量 i 的地址赋给指针 p

5.2.2 通过指针访问值

  • 使用星号（*）解引用指针以访问其指向的值。

xxxxxxxxxx
fmt.Println(*p)                             // 输出：42

5.2.3 指针作为函数参数

  • 通过指针传递参数，可以在函数内部修改原始变量的值。

xxxxxxxxxx
func increment(ptr *int) {
    *ptr = *ptr + 1
}


func main() {
    count := 10
    increment(&count)
    fmt.Println(count)                      // 输出：11
}

5.2.4 new 和 make 函数

  • new(T) 分配了类型为 T 的零值并返回其指针。

  • make(T) 创建了初始化的复合类型，如切片、映射或通道，并返回一个有初始值的 T。

xxxxxxxxxx
p := new(int)                               // 分配一个新的 int 类型的内存，初始值为 0
s := make([]int, 5)                         // 创建长度为 5 的整型切片，初始元素均为 0

5.2.5 指针与结构体

  • 使用指针操作结构体可以减少复制开销，尤其在大型结构体或嵌套结构体中。

xxxxxxxxxx
type Person struct {
    Name string
    Age  int
}


func updatePerson(p *Person) {
    p.Age++
}


func main() {
    person := Person{Name: "Alice", Age: 30}
    updatePerson(&person)
    fmt.Println(person)                     // 输出：{Alice 31}
}

5.2.6 指针接收者方法

  • 结构体方法可以定义为接收者为指针类型，直接修改调用对象。

xxxxxxxxxx
func (p *Person) incrementAge() {
    p.Age++
}


func main() {
    person := Person{Name: "Bob", Age: 40}
    person.incrementAge()
    fmt.Println(person)                     // 输出：{Bob 41}
}

5.2.7 空指针

  • 在 Go 中，指针的零值是 nil，表示没有指向任何内存地址。

xxxxxxxxxx
var ptr *int
fmt.Println(ptr == nil)                     // 输出：true

5.2.8 指针与内存逃逸分析

  • Go 语言的垃圾回收机制会自动管理内存，但过度使用指针可能导致不必要的内存逃逸。

xxxxxxxxxx
func createArray() [10]int {
    return [10]int{}
}


func createSlice() *[]int {
    slice := make([]int, 10)
    return &slice
}

createArray 没有发生逃逸，因为它返回的是栈上的数组。而 createSlice 发生了逃逸，因为它返回的是指向堆内存的指针。

5.2.9 指针数组和指向指针的指针

  • Go 语言支持指针数组和指向指针的指针。

xxxxxxxxxx
var ptrArray [5]*int                        // 指针数组
var ptrToPtr **int                          // 指向指针的指针

5.2.10 向函数传递指针参数

  • 通过引用或地址传参，在函数调用时可以改变其值。

xxxxxxxxxx

}

}

5.4 Go 切片（Slice）

Go 语言中的切片（Slice）是一种基于数组的更高级的数据结构，它提供了一种更加灵活的方式来操作序列数据。

切片并不是数组的某个部分，而是一个轻量级的数据结构，它描述了原数组中一段区域的开始和结束。

5.4.1 切片的基本组成

  • 指向底层数组的指针：切片包含一个指向底层数组的指针。

  • 切片的长度：切片包含一个长度，表示切片中元素的数量。

  • 切片的容量：切片包含一个容量，表示从切片的开始位置到底层数组末尾的元素数量。

5.4.2 切片的创建

  1. 基于数组创建切片：

    xxxxxxxxxx
    arr := [5]int{1, 2, 3, 4, 5}
    slice := arr[1:4]                   // 创建一个新的切片，包含arr的第2到第4个元素

  2. 使用make函数创建切片：

    xxxxxxxxxx
    slice := make([]int, 5)             // 创建一个长度为5的切片，所有元素初始化为0

  3. 使用copy函数复制切片：

    xxxxxxxxxx
    arr := [5]int{1, 2, 3, 4, 5}
    slice := make([]int, 3)
    copy(slice, arr[1:4])               // 将arr的第2到第4个元素复制到slice中

  4. 使用append函数添加元素：

    xxxxxxxxxx
    slice := []int{1, 2, 3}
    slice = append(slice, 4)            // 在切片末尾添加一个元素

5.4.3 切片的属性和方法

  • 长度：使用len()函数获取切片的长度。

  • 容量：使用cap()函数获取切片的容量。

  • 追加：使用append()函数向切片追加元素。

  • 扩展：可以通过索引的方式扩展切片，例如slice[2:] = append(slice[:2], 4)。

  • 截取：可以使用切片操作符[:]来截取切片的一部分。

5.5 Go 范围（Range）

Go 语言中的 “范围”（range）是一种强大的迭代机制，它可以用来遍历数组、切片、映射（map）和通道（channel）。

5.5.1 数组和切片

  • 在数组或切片中使用 range，它会返回每个元素的索引和值。

x
nums := []int{2, 3, 4}
for index, value := range nums {
    fmt.Printf("Index: %d, Value: %d\n", index, value)
}

  • 如果你不需要索引，可以使用下划线 _ 来忽略它。

x
for _, value := range nums {
    fmt.Println("Value:", value)
}

5.5.2 字符串

  • 在字符串上使用 range 会迭代每个 Unicode 码点（rune）。它返回字符的索引和对应的 rune 值。

x
for index, runeValue := range "go" {
    fmt.Println("Index:", index, "Rune:", runeValue)
}

5.5.3 映射

  • 在映射中使用 range 会迭代每个键值对，返回键和对应的值。

x
kvs := map[string]string{"a": "apple", "b": "banana"}
for key, value := range kvs {
    fmt.Printf("%s -> %s\n", key, value)
}

5.5.4 通道

  • 在通道上使用 range 可以迭代通道中发送的值，直到通道被关闭。

xxxxxxxxxx
ch := make(chan int)
go func() {
    for i := 1; i <= 3; i++ {
        ch <- i
    }
    close(ch)           // 一定要记得在发送完数据后关闭通道，否则 range 会一直等待新的数据到来
}()
for value := range ch {
    fmt.Println("Received:", value)
}

六、Go 高级特性

6.1 Go 递归函数

在 Go 语言中，递归函数是指一个函数在其函数体内调用自身。

递归是一种强大的编程技术，它允许函数简洁地表示和解决复杂问题，尤其是那些可以分解为相似子问题的问题。

6.1.1 递归函数的基本构成

递归函数通常包含两个主要部分：

  1. 基本情况（Base Case）：这是递归终止的条件，没有它，递归将无限进行下去，最终导致栈溢出错误。

  2. 递归步骤（Recursive Step）：这是函数调用自身的地方，每次调用都应该朝着基本情况前进。

6.1.2 示例1：计算阶乘

  • 阶乘函数是递归的一个经典例子。n! 表示从 1 到 n 所有整数的乘积。

x
func Factorial(n uint) uint {
    if n == 0 {                                     // 基本情况
        return 1
    }
    return n * Factorial(n-1)                       // 递归步骤
}

6.1.3 示例2：斐波那契数列

  • 斐波那契数列是另一个经典的递归例子，其中每个数字是前两个数字的和。

x
func Fibonacci(n int) int {
    if n <= 1 {                                     // 基本情况
        return n
    }
    return Fibonacci(n-1) + Fibonacci(n-2)          // 递归步骤
}

6.1.4 示例3：二分搜索

  • 二分搜索是一个在有序数组中查找元素的高效算法，它通过递归地将搜索范围减半来工作。

x
func BinarySearch(arr []int, target int, low, high int) int {
    if low > high {                                 // 基本情况
        return -1
    }
    mid := low + (high-low)/2
    if arr[mid] == target {                         // 找到目标
        return mid
    } else if arr[mid] < target {                   // 在右半部分搜索
        return BinarySearch(arr, target, mid+1, high)
    } else {                                        // 在左半部分搜索
        return BinarySearch(arr, target, low, mid-1)
    }
}

6.1.5 示例4：树的遍历

  • 递归也经常用于树结构的遍历，如二叉树的前序、中序、后序遍历。

xxxxxxxxxx
func PreorderTraversal(root *TreeNode) {
    if root != nil {
        fmt.Println(root.Val)                       // 访问根节点
        PreorderTraversal(root.Left)                // 递归遍历左子树
        PreorderTraversal(root.Right)               // 递归遍历右子树
    }
}

6.2 Go 类型转换

在Go语言中，类型转换是一种常见的操作，用于将一个类型的值转换为另一个类型。

  1. 在进行类型转换时，需要注意目标类型是否能够容纳源类型的值，以避免数据溢出或精度丢失。

  2. 显式类型转换是安全的，但类型断言在失败时如果不处理，会导致程序 panic。

  3. 接口类型转换时，应始终先检查底层类型，以确保类型转换的安全性。

6.2.1 显式类型转换

  • 显式类型转换是最基本的类型转换方式，使用 newType(expression)，其中 newType 是目标类型，expression 是要转换的值。

xxxxxxxxxx
var sum int = 17
var count int = 5
var mean float32 = float32(sum) / float32(count)    // 显式转换为 float32 类型

6.2.1 字符串类型转换

  • 字符串与基本数据类型之间的转换通常使用 strconv 包来完成。

xxxxxxxxxx
str := "123"
num, err := strconv.Atoi(str)                       // 将字符串 str 转换为整数 num
if err != nil {
    fmt.Println("转换错误:", err)
} else {
    fmt.Printf("字符串 '%s' 转换为整数为：%d\n", str, num)
}

6.2.1 接口类型转换

  • Go语言中的接口类型可以存储任何类型的值，当需要从接口类型中提取具体的值时，需要进行类型断言。

xxxxxxxxxx
var i interface{} = 10
value, ok := i.(int)                                // 使用类型断言将接口 i 转换为 int 类型
if ok {
    fmt.Println("Value:", value)
} else {
    fmt.Println("Conversion failed")
}

6.2.1 类型别名

  • 类型别名是为现有类型提供一个新名字，虽然它们在语法上相同，但在类型系统中是不同的，因此需要进行类型转换。

xxxxxxxxxx
type myBigInt int64
var a myBigInt = 123                                // 数值常量初始化赋值可以自动转

6.3 Go 接口

在Go语言中，接口（interface）是一种抽象类型，它定义了一组方法签名，任何实现了这些方法的类型都被认为是实现了该接口。

接口在Go中是隐式实现的，这意味着你不需要显式声明一个类型实现了某个接口，只需要实现接口中定义的所有方法即可。

6.3.1 定义接口

  • 例如，定义一个表示几何形状的接口：

xxxxxxxxxx
type Shape interface {              // 这里，Shape 是一个接口，包含 Area 和 Perimeter 两个方法。
    Area() float64
    Perimeter() float64
}

注意：空接口（interface{}）是没有定义任何方法的接口，因此任何类型都实现了至少一个空接口。空接口可以存储任何类型的值。

6.3.2 实现接口

  • 一个类型如果想要实现接口，只需要提供接口中定义的所有方法的实现即可。例如，Rectangle 和 Circle 类型实现 Shape 接口：

x
type Rectangle struct {
    Width, Height float64
}


func (r Rectangle) Area() float64 {
    return r.Width * r.Height
}


func (r Rectangle) Perimeter() float64 {
    return 2 * (r.Width + r.Height)
}
xxxxxxxxxx
type Circle struct {
    Radius float64
}


func (c Circle) Area() float64 {
    return math.Pi * c.Radius * c.Radius
}


func (c Circle) Perimeter() float64 {
    return 2 * math.Pi * c.Radius
}

6.3.3 使用接口

  • 接口类型的变量可以存储任何实现了接口中所有方法的类型的值。例如，可以使用接口作为函数参数来实现多态：

x
func measure(shape Shape) {         // measure 函数接受一个 Shape 接口类型的参数，可以传递任何实现了 Shape 接口的类型
    fmt.Println(shape.Area())
    fmt.Println(shape.Perimeter())
}

  • 然后可以这样调用：

xxxxxxxxxx
rect := Rectangle{Width: 5, Height: 10}
circle := Circle{Radius: 5}


measure(rect)                       // 输出矩形的面积和周长
measure(circle)                     // 输出圆形的面积和周长

6.3.4 类型断言

  • 接口类型变量中存储的具体值可以通过类型断言来访问：

xxxxxxxxxx
value, ok := someInterface.(Type)   // value 是存储在接口中的值，如果类型断言成功，ok 为 true；否则为 false。

 

 

七、Go 错误处理与并发

7.1 Go 错误处理

错误处理主要依赖于 error 接口，这是一个内建的接口，任何实现了 Error() string 方法的类型都可以作为错误使用。

错误处理通常涉及两个步骤：返回错误和检查错误。

  1. 返回错误：当一个函数可能遇到问题时，它会返回一个 error 类型的值。按照惯例，错误值是函数的最后一个返回值。

  2. 检查错误：调用可能返回错误值的函数时，应该将错误值赋给一个变量，并与 nil 比较，以确定是否发生了错误。

xxxxxxxxxx
result, err := someFunction()
if err != nil {
    // 处理错误
    log.Fatal(err)
}

  • Go 1.13 版本引入了对错误包装的支持，可以使用 fmt.Errorf 与 %w 动词来创建包装了另一个错误（wrap）的新错误。

xxxxxxxxxx
err := someFunctionThatMightFail()
if err != nil {
    return fmt.Errorf("an error occurred: %w", err)
}

  • 此外，errors.Is 和 errors.As 函数可以用来检查错误链中是否存在特定的错误类型。

 

7.2 Go 并发编程

Go语言的并发编程主要依赖于两个核心概念：goroutine 和 channel。

Goroutine 是 Go 语言中的轻量级线程，它允许程序以并发的方式执行任务。Channel 则用于在 goroutine 之间安全地传递数据。

  1. Goroutines：Goroutines 是 Go 中实现并发的轻量级线程。创建一个新的 goroutine 只需要在函数调用前加上 go 关键字。

xxxxxxxxxx
go someFunction()

  • 当需要等待多个 goroutines 完成时，可以使用 sync.WaitGroup。

xxxxxxxxxx
var wg sync.WaitGroup
wg.Add(1)


go func() {
    defer wg.Done()
    // do work
}()
wg.Wait()   // 等待所有 goroutine 完成

  1. Channels：Channels 用于在 goroutines 之间安全地传递数据。它们可以是带缓冲的或不带缓冲的。

xxxxxxxxxx
ch := make(chan int)                    // 无缓冲 channel
ch := make(chan int, 10)                // 缓冲大小为 10 的 channel

  • Channels 是双向的，可以用于在 goroutines 之间发送和接收数据。

xxxxxxxxxx
ch <- v                                 // 发送数据到 channel
v := <-ch                               // 从 channel 接收数据

  • 当需要在多个 channel 操作之间进行选择时，select 语句非常有用。

xxxxxxxxxx
select {
    case v := <-ch1:                    // 处理 ch1
    case v := <-ch2:                    // 处理 ch2
}
